👤 Users Table

  • UUID primary key
  • Unique email constraint
  • Hashed password (bcrypt)
  • Has many: transactions, categories, budgets

💵 Transactions Table

  • UUID primary key
  • Decimal amount (12,2 precision)
  • Type: INCOME or EXPENSE
  • Indexed on (userId, date)

📁 Categories Table

  • UUID primary key
  • Type: INCOME or EXPENSE
  • Default + custom categories
  • Icon and color customization

💰 Budgets Table

  • UUID primary key
  • Monthly budget limits
  • Unique constraint on (userId, categoryId, month, year)
  • Linked to categories
